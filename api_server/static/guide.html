효과: 세심하게 계획한 일정을 현장에서 바로 확인

운동 루틴 관리

이 대화를 저장해줘. 제목: "홈트 30분 루틴 - 상체 집중", 태그: "운동, 홈트, 상체"

💬 다음 운동 날

효과: 체계적인 운동 루틴으로 꾸준한 건강 관리

책/영화 감상 기록

저장해줘. 제목: "인터스텔라 - 시간과 차원에 대한 토론", 태그: "영화, SF, 명작"

💬 친구에게 SF 영화 추천할 때

🔧 MCP 서버 설정

1

MCP 서버 연결하기

핵심 단계

사용하시는 AI 환경에 맞는 방법을 선택하세요:

CLI 터미널

터미널에서 다음 명령어를 실행하여 Pensieve MCP 서버를 추가하세요: 

claude mcp add --transport sse pensieve https://pensieve.borihopang.com/sse

명령어 실행 후 바로 사용 가능합니다.

참고: Claude Code를 재시작할 필요 없이 HTTPS를 통해 안전하게 연결됩니다.

웹/데스크톱 앱 Pro/Team 전용

Claude 웹 또는 데스크톱 앱에서 커스텀 커넥터로 연결할 수 있습니다:

설정 방법

  1. Claude 설정 커넥터 메뉴로 이동
  2. "커스텀 커넥터 추가" 클릭
  3. 아래 URL 입력:
https://pensieve.borihopang.com/sse

URL만 입력하면 자동으로 연결됩니다!

OpenAI ChatGPT 개발자 모드 필요

ChatGPT에서도 MCP 서버를 연결할 수 있습니다:

설정 방법

  1. ChatGPT 설정 메뉴 열기
  2. 연동 앱 및 커넥터 선택
  3. 개발자 모드 활성화
  4. "만들기" 클릭
  5. 아래 URL 입력:
https://pensieve.borihopang.com/sse

설정 완료 후 ChatGPT에서 Pensieve 도구를 사용할 수 있습니다!

2

계정 생성 및 로그인

AI에게 회원가입 또는 로그인을 요청하세요:

처음 사용시 (회원가입):

Pensieve에 회원가입해줘. 이메일: your@email.com, 비밀번호: yourpassword

이미 계정이 있다면 (로그인):

Pensieve에 로그인해줘. 이메일: your@email.com, 비밀번호: yourpassword

웹 대시보드(메인 페이지)에서도 회원가입이 가능합니다.

3

Pensieve 사용하기

로그인 후 다양한 기능을 사용할 수 있습니다:

💾 대화 저장:

이 대화를 Pensieve에 저장해줘. 제목: "프로젝트 설정", 태그: "setup, config"

📋 대화 목록 조회:

Pensieve에 저장된 대화 목록을 보여줘

🔍 대화 검색:

Pensieve에서 "React"가 포함된 대화를 검색해줘

📥 대화 불러오기:

Pensieve에서 대화 ID "abc123"을 불러와줘

모든 대화는 클라우드에 안전하게 저장되어 언제든지 다시 불러올 수 있습니다!

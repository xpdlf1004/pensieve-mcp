대시보드 사용 가이드 MCP 설정

🔧 MCP 서버 설정

Claude Code와 Pensieve를 연결하는 방법입니다

1

Claude Code 설치

먼저 Claude Code가 설치되어 있어야 합니다. 아직 설치하지 않았다면 공식 웹사이트에서 설치 방법을 확인하세요.

✓ macOS, Windows, Linux 지원

✓ 터미널 기반 AI 코딩 도구

✓ MCP (Model Context Protocol) 지원

2

Pensieve 사용자 계정 생성

Pensieve를 사용하려면 먼저 사용자 계정을 생성해야 합니다:

1. 이 대시보드의 메인 페이지로 이동

2. "회원가입" 버튼 클릭

3. 이메일과 비밀번호로 계정 생성

생성한 계정 정보는 다음 단계에서 사용됩니다.

3

Claude Code에 MCP 서버 추가

터미널에서 다음 명령어를 실행하여 Pensieve MCP 서버를 추가하세요: 

claude mcp add --transport sse pensieve https://pensieve.borihopang.com/sse

인터넷이 연결된 어디서나 사용할 수 있습니다.

참고:

  • 명령어 실행 후 MCP 서버가 자동으로 추가됩니다
  • Claude Code를 재시작할 필요 없이 바로 사용 가능합니다
  • HTTPS를 통해 안전하게 연결됩니다
4

Pensieve 로그인

Claude Code 터미널에서 Pensieve 명령어를 사용하여 로그인하세요:

# 회원가입한 이메일과 비밀번호로 로그인
Pensieve에 로그인해줘. 이메일: your@email.com, 비밀번호: yourpassword

또는 register 명령어로 바로 가입:

Pensieve에 회원가입해줘. 이메일: your@email.com, 비밀번호: yourpassword
5

Pensieve 사용하기

로그인 후 Claude Code에서 Pensieve의 다양한 기능을 사용할 수 있습니다:

💾 대화 저장:

이 대화를 Pensieve에 저장해줘. 제목: "프로젝트 설정", 태그: "setup, config"

📋 대화 목록 조회:

Pensieve에 저장된 대화 목록을 보여줘

🔍 대화 검색:

Pensieve에서 "React"가 포함된 대화를 검색해줘

📥 대화 불러오기:

Pensieve에서 대화 ID "abc123"을 불러와줘

모든 대화는 클라우드에 안전하게 저장되어 언제든지 다시 불러올 수 있습니다!

❓ 문제 해결

MCP 서버 연결 확인

claude mcp list

등록된 MCP 서버 목록을 확인할 수 있습니다.

로그인이 안 될 때

  • 이메일과 비밀번호가 정확한지 확인
  • 먼저 웹 대시보드에서 회원가입을 완료했는지 확인
  • 서버 URL이 올바른지 확인 (로컬 또는 운영)

MCP 서버 제거 방법

claude mcp remove pensieve

등록된 MCP 서버를 제거합니다.
